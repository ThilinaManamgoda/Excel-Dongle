Follow these instructions

1. Configure the RxTx API
2. Install Maven (https://maven.apache.org/install.html)
3. Go to Libs folder and execute installLibs.bat file 
4. Go to Excel-Dongle folder and open CMD and type "mvn install"
5. Go to target folder and get the Dongle-Excel.one-jar.jar file
6. open config.txt file and follow the instructions 
7. put above  jar file and the config.txt in a folder and execute "java -jar Dongle-Excel.one-jar.jar" in a CMD
